Q: 从输入URL到页面显示发生了什么?
A: 在浏览器输入url后，浏览器主进程处理输入信息后开始导航，与网络进程通信，准备发起url请求，此时并不会直接发起请求，会首先查找本地缓存资源，如果有就返回资源给浏览器，没有的话就发起url请求。
总体来说分为以下几个过程:
DNS 解析:将域名解析成 IP 地址
TCP 连接：TCP 三次握手
发送 HTTP 请求
服务器处理请求并返回 HTTP 报文
浏览器解析渲染页面
断开连接：TCP 四次挥手
